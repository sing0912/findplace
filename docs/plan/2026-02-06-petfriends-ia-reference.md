**작성일:** 2026-02-06

## 1. 사이트 구조 개요

펫프렌즈는 **3개의 독립 어드민**과 **1개의 고객용 서비스**로 분리되어 있음.

| 구분 | 도메인 | 역할 |
|------|--------|------|
| 고객용 (B2C) | m.pet-friends.co.kr | 쇼핑몰, 커뮤니티, 마이페이지 |
| 판매자 어드민 | seller.pet-friends.co.kr | 입점 판매자 관리 |
| 발주 어드민 | supply.pet-friends.co.kr | 공급/발주 관리 |
| 본사 어드민 (HQ) | hq.pet-friends.co.kr | 플랫폼 전체 관리 |

## 2. 고객용 서비스 (B2C) IA

### 2.1 글로벌 네비게이션 (상단 탭바)

```
┌──────────────────────────────────────────────────────────┐
│ 홈메인 │ 맘마샘플 │ 베스트 │ 신상품 │ 단독판매 │ 이벤트 │ 검색 │ 집사생활 │
```

| # | 메뉴 | 경로 | 설명 |
|---|------|------|------|
| 1 | 홈메인 | /main/home | 메인 홈 (배너, 추천, 카테고리) |

## 4. PetPro 적용 시사점

### 4.1 고객용 사이드바 참고

펫프렌즈의 고객 네비게이션은 **쇼핑 중심**:
- 상단: 상품 탐색 (베스트, 신상품, 이벤트)
- 하단: 핵심 기능 (홈, 카테고리, 검색, 장바구니, 마이페이지)

**PetPro 고객 메뉴 (장례 서비스 특화):**
- 홈, 내 주변 장례식장, 예약 내역, 주문 내역, 추모관, 마이페이지

### 4.2 어드민 분리 참고

펫프렌즈는 어드민을 **3개 도메인으로 완전 분리**:
- 본사(HQ): 플랫폼 전체 관리
- 판매자(Seller): 입점 업체 자체 관리
- 발주(Supply): 공급/물류 관리

**PetPro 어드민 구조:**
- 플랫폼 관리자 (/admin): 전체 시스템 관리
- 장례업체 관리자: 예약/일정/봉안당 관리 (향후 분리 고려)
- 공급사 관리자: 상품/재고/주문/정산 관리 (향후 분리 고려)
