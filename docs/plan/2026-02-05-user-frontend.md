## 1. 개요

### 1.1 목적
Figma 디자인 및 신규 설계를 기반으로 User 도메인의 프론트엔드 화면을 구현합니다.

### 1.2 Figma 참조
- **URL:** https://www.figma.com/design/mXrXb73tJYn0qzE9jKgEUv/펫프로-와이어프레임

### 1.3 구현 범위 (오늘)

| # | 기능 | 화면 | Figma 존재 |
|---|------|------|-----------|
| 1 | 로그인 (카카오/네이버/구글) | 로그인 시작, 소셜 로그인 | ✅ |
| 2 | 아이디 찾기 (휴대폰 인증) | 아이디 찾기, 인증, 결과 | ✅ |
| 3 | 비밀번호 재설정 | 비밀번호 찾기, 재설정 | ✅ |
| 4 | 회원가입 | 약관동의, 정보입력, 완료 | ✅ |
| 5 | 마이페이지 | 마이페이지 메인 | ⏳ 구현 후 Figma 추가 |
| 6 | 회원 정보 수정 | 프로필 수정 | ⏳ 구현 후 Figma 추가 |
| 7 | 문의 게시판 | 문의 목록, 작성, 상세 | ⏳ 구현 후 Figma 추가 |
