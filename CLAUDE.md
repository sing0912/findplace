# Claude Code 프로젝트 지침

**모든 규칙과 지침은 `AGENTS.md`에 정의되어 있습니다.**
**작업 시작 전 반드시 `AGENTS.md`를 먼저 읽고 참조하세요.**

```
AGENTS.md  ← 프로젝트 메인 지침 (필수 참조)
```

## 커밋 메시지 필수 규칙 (절대 준수)

- **모든 커밋 메시지는 반드시 한글로 작성**
- **영문 커밋 메시지 절대 금지** (feat:, fix:, chore:, refactor:, docs: 등 금지)
- **형식**: `[타입] 한글 제목` + 한글 본문
- **타입**: 기능, 수정, 리팩토링, 문서, 테스트, 설정, 보안, 변경, 추가, 삭제
- **PR 제목/본문도 반드시 한글로 작성**

### 예시
```
[기능] 마이페이지 약관/정책 목록 구현
[수정] 예약 날짜 유효성 검사 오류 해결
[테스트] 고객/파트너 홈 E2E 테스트 추가
[설정] Docker 환경 변수 추가
[문서] API 명세서 업데이트
```
